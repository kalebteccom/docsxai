# docsxai brand kit

Source of truth for the docsxai mark and icon set. Everything raster is
generated by one script: `docsxai.py` (cairosvg + pillow).

## The mark

A 300-degree loop with the gap centered on an escaping line: two document
lines, the top one running out through the opening. The replay loop emits
the doc - calibrate once, replay forever, docs come out. Third sibling in
the xai family: browxai's inner glyph is a navigation cursor-kite (violet
aurora), remotxai's is a signal ring with anchor dots (teal signal),
docsxai's is the loop-and-line (amber ember). All three share the same
canvas geometry, glass treatment, and generation pipeline.

## Files

| File                                                                   | What it is                                       |
| ---------------------------------------------------------------------- | ------------------------------------------------ |
| `docsxai-mark.svg`                                                     | currentColor mark (use in docs/UI)               |
| `docsxai-mark-black.svg` / `-white.svg`                                | fixed-ink marks                                  |
| `docsxai-favicon.svg`                                                  | amber tile + white mark (the favicon source)     |
| `favicon-16/32/48.png`, `favicon.ico`, `apple-touch-icon.png`          | rasterized favicon set                           |
| `docsxai-ember-fullbleed-1024.png`                                     | full-bleed app-icon master                       |
| `docsxai-glass-ember-1024.png`                                         | squircle glass icon                              |
| `icon-composer-layers/`                                                | background + mark layers for Apple Icon Composer |
| `docsxai-avatar-512.png`                                               | GitHub org/app avatar                            |
| `ember-preview.png`, `favicon-tile-preview.png`, `xai-family-trio.png` | review sheets                                    |

The 16 px favicon uses a simplified variant (heavier strokes, single line)
so the mark stays legible at browser-tab size.

## Regenerating

```
pip install cairosvg pillow
python3 docsxai.py
```

The family trio sheet expects the sibling masters at
`../../browxai/brand/` and `../../remotxai/brand/` (override with
`BROWXAI_GLASS` / `REMOTXAI_GLASS`).

## Where these are used

- `website/public/` - favicon.svg, favicon-16/32.png, favicon.ico, apple-touch-icon.png
- `website/src/assets/docsxai-tile.svg` - header logo (tile + sheen)
- `website/scripts/gen-og.mjs` - the OG card draws the same mark paths
